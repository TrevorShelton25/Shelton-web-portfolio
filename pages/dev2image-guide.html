There are three main types of images located on the web: JPEG, PNG, and GIF. Below is a list of each type of image and their purpose.


JPEG PNG GIF
JPEG is an important image style because of its ability to produce high resolution images in small file sizes Images saved in PNG files have higher compression rates which allow files to download quicker GIF is a suitable file type for saving moving pictures. These pictures are saved in several different frames that are played in a loop
